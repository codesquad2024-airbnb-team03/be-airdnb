### 협업 규칙
- 스크럼 시간: 출근 직후, 필요할 때마다 🤗
- BE: dev-be에 계속 머지, FE: master에 계속 머지
- GitHub Issues를 사용해서 Task 관리하기
- 모든 작업 전에는 이슈를 생성해서 커밋 메시지에 이슈번호 적기
- 서로 공유할 내용 노션에 계속 정리하기
